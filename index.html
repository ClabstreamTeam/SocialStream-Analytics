FAQ

Questions buyers typically ask

Does SocialStream expose API tokens to the browser?

No. Tokens are kept server-side in deployment environment variables and called through API endpoints.

Can I export reports without marketing page content?

Yes. Reports are generated via a dedicated report page that contains report content only.

Can my team compare multiple TikTok profiles?

Yes. The workspace supports profile-level filtering and comparison within the same analysis context.

Can we start with a demo before committing?

Yes. Use Start Demo to load example data and preview the full workflow immediately.
